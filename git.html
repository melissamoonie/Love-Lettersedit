My Dearest Git,

Thanks for finally giving me your diGITs (wow that was pretty lame). Let's git in it and create a directory! Maybe we can configure sometime... and set up some aliases. I hope you're willing to commit. I need to rely on you to commit all of my edits. Let's check your status because you're the only one who gits me. You push me in such useful ways and you allow me to easily share myself with the world. Add . me ;) Someday I'll gitcha something nice.
git config -- global user.name "My Name"
git config --global user.email
(Necessary step for successful logon)
git init
(To create a new Git repository/ initialize a new empty repository.)
git status
(Check the current state of the repository and the working directory.)
git add (filename)
(Adds a new file.)
git push
(Pushes to the web.)
git reset
(Necessary if you decide to unstage changes.)
git checkout -b (branchname)
(Enables the creation of a git branch.)
git branch #
(Checks which branch the user is currently on.)
git merge (branchname)
(Merges a branch onto the master branch.)
